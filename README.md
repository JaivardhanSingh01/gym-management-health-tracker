# 🏋️ Gym Management & Health Tracker

A full-stack web application for managing gym members and tracking individual health progress — built with HTML, CSS, JavaScript, and Google Firebase.

---

## 🔗 Live Demo

> 🚧 Coming soon — will be deployed on Netlify/GitHub Pages

---

## 📋 Features

- 👤 **User Authentication** — Secure login and registration via Google Firebase Auth
- 🏠 **Admin Dashboard** — Manage all gym members from a single control panel
- 📊 **Health Tracking** — Monitor individual member health metrics and progress
- 📱 **Responsive Design** — Works seamlessly on desktop and mobile devices
- ☁️ **Real-time Database** — Live data sync using Google Firebase Firestore
- 🔒 **Role-based Access** — Separate views for admin and regular members

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Structure and layout |
| CSS3 | Styling and responsive design |
| JavaScript (ES6) | Frontend logic and interactivity |
| Google Firebase | Authentication, Firestore database, cloud storage |
